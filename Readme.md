# My Portfolio Website | 個人自我紹介サイト

一個純前端的自我介紹型作品集網站。主題包含：個人興趣（501軍団/星戰Cosplay）、喜愛遊戲、電影、音樂、美食等。每頁主題設計突出，結合視差滾動、卡片式照片牆、動態動畫等現代Web效果。

---

## 🚀 項目亮點

- **純 HTML + CSS**（無前端框架/無打包）
- **響應式設計**，適合桌面端瀏覽
- **視差滾動**與**進場動畫**（AOS, Locomotive Scroll）
- **卡片式/照片牆布局**，主題感明確
- **固定側邊導航**，所有頁面共用
